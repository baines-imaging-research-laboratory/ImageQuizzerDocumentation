# Session

## Specs

| ||
|---|---|
| **Name** | Session |
| **Classification** | element |
| **Parent** | - |
| **Required** | yes |
| **Syntax** | <Session\> |

## Description
Session is the root element in the XML tree. It is must exist in the XML quiz file as the outermost element. 
It encompasses all the child Page elements defined for the study.


## Example
```
<Session>
	<Page>
		...
	</Page>
	<Page>
		...
	</Page>
</Session>
```
