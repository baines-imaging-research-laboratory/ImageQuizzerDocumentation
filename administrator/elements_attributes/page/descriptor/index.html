    Descriptor

    Specs

    Details
    Name Descriptor
    Classification attribute
    Parent <Page>
    Required no
    Syntax Descriptor="option"
    Options character string used in compbination with ID to identify the Page

    See also: Page, ID

    Description

    Descriptor is not a required attribute for the Page element. It is used in combination with the ID attribute (required) to identify the Page element.

    The ID_Descriptor string combination appears as an identifier in the Slicer layout when displaying the images. It is also used as part of the subfolder names (prefixed by the PageGroup number) created in the User's results folder when capturing contours and measurement lines that the user creates during the quiz.
